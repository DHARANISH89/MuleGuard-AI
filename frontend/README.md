# FR (frontend)

Simple React frontend app (created with Create React App).

## Description

This repository contains a small React frontend located in the `src` folder. It includes a build output under `build/` and a public folder for static assets.

## Prerequisites

- Node.js (recommended >= 16)
- npm (recommended >= 8) or yarn

## Install

From the project root run:

```bash
npm install
```

## Run (development)

```bash
npm start
```

This runs the app in development mode at http://localhost:3000 by default.

## Build (production)

```bash
npm run build
```

The production-ready files will be in the `build/` directory.

## Test

```bash
npm test
```

## Project structure (important files)

- `src/` — application source code
- `public/` — static public files
- `build/` — production build output (generated)
- `package.json` — project metadata and dependencies
